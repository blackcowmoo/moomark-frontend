# moomark-frontend

Next.js 프레임 워크 활용 마크다운 기반 커뮤니티

## Run in Local

```bash
yarn
yarn dev
```

## lint

```bash
yarn lint
yarn lint --fix
```

## test

```bash
yarn test
yarn test -u # update snapshot
yarn test:coverage # coverage
```

## Stack

- next.js
- react
- scss
- jest
- typescript
- recoil
- apollo
